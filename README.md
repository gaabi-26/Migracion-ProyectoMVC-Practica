Descripción breve
-----------------
Proyecto práctico para el curso C# Nivel 4 [MVC + EF]: migración de una aplicación de consola (capas dominio y negocio) a una aplicación web en ASP.NET Core MVC con vistas Razor. Permite gestionar un catálogo de discos musicales (CRUD) con los modelos Disco, Estilo y TipoEdicion.

Tecnologías
----------
- .NET 9, C# 13
- ASP.NET Core MVC (vistas Razor)
- Bootstrap para la interfaz
- Capas: dominio, negocio, presentación (MVC)
